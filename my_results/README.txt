In this folder, you will find the following:
	- statlearning-kaggle.pdf: a report on what I did in this project
	- statlearning-kaggle.py: a script that reproduces the results

-> Running statlearning-kaggle.py
	1- Download statlearning-sjtu-2019.zip and put it in the same folder. 
	2- Have Anaconda installed, it contains all the necessary libraries for data processing and ML
	3- run python 3: $ python statlearning-kaggle.py

PS: XGBoost will not install for non UNIX systems.
